## What is seeleseek?

seeleseek is a native macOS client for the [Soulseek](https://www.slsknet.org/) peer-to-peer file sharing network. The app is written in Swift and SwiftUI. It does not use Electron or web views.

## System Requirements

- **macOS 15.6** or later
- A connection to the internet
- A Soulseek account (you can make one during setup)

## Installation

1. Download the latest `.pkg` file from the [downloads page](/).
2. Open the `.pkg` file and obey the installer instructions.
3. Start seeleseek from the Applications folder or from Spotlight.

At the first start, macOS can ask you for permission to open the app. This occurs because the app came from the internet.

## Make an Account

If you do not have a Soulseek account, enter a new username and password on the login screen. The server makes the account automatically if the username is not in use.

## First Steps

After you connect, do these steps:

1. **Search for music.** Type a query in the search bar and press Enter. Results come from other users on the network.
2. **Download a file.** Click a search result to see the details. Then click the download button.
3. **Share your files.** Open Settings (⌘9) and add folders to share. Shared files keep the network healthy.
