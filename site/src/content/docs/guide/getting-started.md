## What is seeleseek?

seeleseek is a native macOS client for the [Soulseek](https://www.slsknet.org/) peer-to-peer file sharing network. It's built with Swift and SwiftUI, so it feels right at home on your Mac — no Electron, no web views.

## System Requirements

- **macOS 15 Sequoia** or later
- An active internet connection
- A Soulseek account (you can create one during setup)

## Installation

1. Download the latest `.pkg` from the [downloads page](/)
2. Open the `.pkg` and follow the installer prompts
3. Launch seeleseek from Applications or Spotlight

On first launch, macOS may ask you to confirm you want to open the app since it was downloaded from the internet.

## Creating an Account

If you don't have a Soulseek account yet, you can use any username and password on the login screen — the server will automatically create the account if the username isn't taken.

## First Steps

Once you're connected, here's what to try first:

1. **Search for music** — Type a query in the search bar and press Enter. Results stream in from other users on the network.
2. **Download a file** — Click a search result to see details, then hit the download button.
3. **Set up sharing** — Go to Settings (⌘9) and add folders to share. Sharing is how the network stays healthy.
